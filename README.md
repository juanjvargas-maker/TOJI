# TOJI
TOJI is a modular autonomous robotics platform designed for both competitive robotics and smart logistics applications. The project combines autonomous navigation, obstacle detection, teleoperation, and object interaction into a compact and adaptable robotic system.

Originally inspired by mini sumo robotics, TOJI evolved into a multifunctional platform capable of:

- Autonomous mini sumo combat
- Teleoperated mini soccer
- Warehouse/logistics simulation
- Box pushing and transport tasks
- Obstacle avoidance and intelligent navigation
  
The robot is powered by an ESP32 and integrates VL53L0X ToF sensors, DRV8833 motor drivers, high-speed DC motors, and a modular PET chassis optimized for rapid adaptation between different operational modes.

Features

- Autonomous operation
- Wireless teleoperation
- Modular hardware architecture
- Real-time obstacle detection
- Multi-sensor integration
- Compact high-torque drivetrain
- Logistics-inspired robotic behavior

Technologies

- ESP32
- VL53L0X
- DRV8833
- FreeRTOS / Embedded C++
- Wireless control systems
- SLAM-inspired navigation concepts

Vision

TOJI aims to bridge educational robotics, competitive robotics, and Industry 4.0 concepts by creating an accessible low-cost platform inspired by real autonomous warehouse robots and industrial AMRs.
